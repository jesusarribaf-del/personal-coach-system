---
description: Inicia una práctica guiada de meditación o respiración según estado actual.
argument-hint: [duración disponible y estado: estrés, ansiedad, calma de mantenimiento, sueño]
---

Delega en `meditation-mindfulness-guide`.

Contexto: $ARGUMENTS

Para el subagente:
1. Leer `memory/meditation-journal.md` (últimas entradas) y `memory/sleep-energy-log.md` si la consulta toca estrés o sueño.
2. Diagnosticar brevemente el estado.
3. Proponer práctica adecuada (no la misma para crisis aguda que para mantenimiento):
   - **Crisis aguda** → respiración 4-7-8 o suspiro fisiológico, 3-5 ciclos.
   - **Estrés sostenido** → 10 min anapana / coherencia cardíaca.
   - **Calma de mantenimiento** → 10-15 min vipassana / atención abierta.
   - **Antes de dormir** → escaneo corporal + gratitud.
   - **Por la mañana** → respiración + intención del día.
4. Guiar paso a paso, párrafos cortos, ritmo lento.
5. Cierre con una idea breve y, si aporta, una pregunta.
6. Proponer entrada para `memory/meditation-journal.md`:

```
[fecha] — práctica: [...] — duración: [...] — estado antes: [...] — estado después: [...] — observación: [...]
```
