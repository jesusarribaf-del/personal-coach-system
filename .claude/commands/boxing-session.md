---
description: Diseña una sesión de boxeo concreta compatible con el plan de fuerza vigente.
argument-hint: [duración disponible, lugar, objetivo: técnica/condicionamiento/sparring]
---

Delega en `strength-conditioning-coach`.

Contexto del usuario: $ARGUMENTS

Pasos para el subagente:
1. Leer `memory/body-training-profile.md` (lesiones, manos, hombros, cervical) y el plan de fuerza vigente para no canibalizar recuperación.
2. Confirmar: duración total, material disponible (saco, manoplas, cuerda, espacio), nivel actual.
3. Diseñar la sesión:

```
SESIÓN DE BOXEO — [duración] — objetivo [técnica / condicionamiento / mixto]

CALENTAMIENTO ESPECÍFICO ([X min])
- Movilidad cervical, hombro, cadera
- Cuerda o sombra ligera

BLOQUE TÉCNICO ([X min])
- [drills concretos: jab, jab-cross, esquivas, footwork]

BLOQUE ACONDICIONAMIENTO ([X min])
- [intervalos saco / sombra / cuerda con tiempos exactos]

ENFRIAMIENTO ([X min])
- Movilidad hombros, cadera, respiración nasal

REGLAS DE SEGURIDAD
- [técnica antes que potencia]
- [vendaje]
- [si hay molestia mano/hombro: parar]

INTEGRACIÓN CON FUERZA
- Hoy/mañana entrenas: [...]
- Ajuste recomendado: [...]
```
