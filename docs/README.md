# docs/

Documentación de referencia, plantillas y ejemplos.

```
docs/
  references/
    training/       # referencias técnicas de entrenamiento
    nutrition/      # referencias nutricionales
    meditation/     # referencias contemplativas
    finance/        # referencias financieras
    productivity/   # referencias de productividad
    strategy/       # modelos mentales, decisiones
  templates/        # plantillas reutilizables (recetas, planes, journaling)
  examples/         # ejemplos de salidas reales del sistema
```

## Uso

- `references/`: material que los agentes pueden citar o usar como base.
- `templates/`: estructuras reutilizables — los comandos pueden referenciarlas.
- `examples/`: ejemplos para entender qué formato espera el sistema.

## Convención

Cada subcarpeta puede tener su propio `README.md` con índice de contenidos.
