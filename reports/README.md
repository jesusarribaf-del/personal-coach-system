# reports/

Reportes generados por las revisiones del sistema.

```
reports/
  weekly/    # AAAA-WW.md — generado por /weekly-review
  monthly/   # AAAA-MM.md y AAAA-MM-finanzas.md — generado por /monthly-review y /spending-audit
  quarterly/ # AAAA-Qn.md — revisiones trimestrales y planes estratégicos
```

## Convenciones

- `weekly/AAAA-WW.md` — usar número de semana ISO.
- `monthly/AAAA-MM.md` — mes calendario.
- `quarterly/AAAA-Qn.md` — trimestre natural (Q1=ene-mar, Q2=abr-jun…).

## Quién escribe

- `/weekly-review` → `reports/weekly/`
- `/monthly-review` → `reports/monthly/`
- `/spending-audit` → `reports/monthly/AAAA-MM-finanzas.md`
- `/strategy-plan` o revisión trimestral → `reports/quarterly/`

Los reportes son **históricos**: no se editan tras crearse, salvo correcciones obvias.
