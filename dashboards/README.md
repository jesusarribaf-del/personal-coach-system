# dashboards/

Vistas agregadas para escaneo rápido del estado.

```
dashboards/
  personal/    # vista global integrada (cuerpo, mente, dinero, sueño)
  training/    # marcas, volumen semanal, progresión
  finance/     # gasto vs presupuesto, ahorro, colchón
  nutrition/   # peso, adherencia, proteína, gasto comida
```

## Para qué sirven

Resumir mucho en poco. Un dashboard se mira en 30 segundos y dice si estás dentro o fuera.

## Formato

Markdown con tablas y semáforos (✅ / ⚠ / 🔴). Pueden generarse manualmente o por comando dedicado.

## Ejemplo de header

```
# Dashboard personal — actualizado AAAA-MM-DD

## Semáforos globales

- Físico: ✅
- Nutrición: ⚠
- Mente: ✅
- Dinero: 🔴
- Sueño: ✅
- Disciplina: ⚠
```

Los dashboards se actualizan en revisiones semanales/mensuales.
