# 01-perfil-inicial/

Setup único. Una vez rellenado y volcado a `memory/`, esta carpeta queda como archivo.

## Qué subir aquí

- **Texto libre** con tus datos personales (respuesta al cuestionario inicial)
- **Export de Jefit** — historial completo o marcas actuales (CSV o captura)
- **Export/captura de Apple Salud** — peso, altura, FC reposo
- **Analítica médica** reciente (PDF o foto) si quieres que la tenga en cuenta
- **Cualquier plan de entreno o dieta** que hayas seguido antes

## Qué NO subir aquí

Capturas diarias de AutoSleep o entrenos → van a `03-descanso/` y `04-entrenos/`
